    Programming Languages

    An Article by Maninder Kaur

  • Introduction

    When I started programming just 1 year ago, the biggest issue I faced was picking a programming language. Many say “It doesn’t matter what you pick, you just need to learn!”. So I decided to ask some programmers what each programming language is good for. Here is a list of programming languages and their uses. As someone who started, a now regrets the choices I made, I don’t want anyone else to have to go through with that.

  • HTML/CSS

    Two languages that I believe that every programmer should know. These two are a great introduction into the world of programming. The two languages highlight syntax, tags and other important concepts. HTML is the bones of the website, while the CSS is the skin and meat. HTML is the content, while CSS is used to accessorize. CSS can come in different forms too like Sass. Try making a basic website just to learn the basics.

  • JavaScript

    While HTML and CSS just create a basic website, we can create one that one can interact with. If HTML is the bones, CSS is the skin and meat, then JavaScript is movement. Javascript can also be used to thoroughly enhance the CSS, for example by using React,js or Three.js.

  • Python

    Python (the programming language, not the animal) is a very versatile language that has uses across all platforms. If you want a way for your website to interact, you could use frameworks like Django and Flask to create dynamic websites. It also has a hand in computing and data science, filled with libraries to manipulate and analyze data. In other words, you can you it for ALMOST anything.

  • C/C++

    All programming languages derive their syntax from these two programming languages, known as the Big Two. Anything related with networks and operating systems are largely programmed using these two languages. Some of your favorite video games are made by C/C++, and when utilize microcomputers, these two languages harbor the power to be used. Ardunio and Raspberry PIs are mainly run on C/C++.

  • Java

    Java is also extremely versatile and user friendly, as it allows people a leg up into other languages. Certain website frameworks require Java servers, to store and manipulate the data. Java is most well known for app development, where it is often used. Majority of scientific programming is also done using Java. When large data needs to be analyzed or manipulated, Java has extensive libraries

  • PHP

    Often used on websites alongside HTML, CSS, and JavaScript. It can also enhance the interactiveness of a website just as JavaScript does, while working alongside the server-side to receive and use data. The biggest examples of PHP use is a login or signup, and PHP will get the data, compare it or add it to the server and allow you into the website.

  • SQL

    SQL is used mainly from website servers. It stores the data that can be compared, verified, manipulated or added into it. SQL doesn’t just stop at web development, but can be implemented into many other platforms, such as microcomputers and apps. Instead of just created variables to store your data, you can store these variables in a server using SQL, putting all the data in one place, and implement multiple servers by category.

  • Go

    Go ore Golang, is a programming language created by the developers at Google. Used mainly for website development, it can be used in game development and in cryptocurrency. The biggest reason it was created was for system-level programming to construct networks, proxies, etc. It’s built in support for blockchain allows many websites to accept Bitcoin or other forms of cryptocurrency.

  • Rust

    Rust, developed by Mozilla, gained popularity in the game development community, for its performance benefits and memory safety. Later adopted into other platforms such as web development and Network programming, the memory safety, concurrent and performance make it a good programming language for system programming, network services, blockchain, command line and cybersecurity.

  • Ruby

    Ruby is a versatile object-oriented programming language. You might have heard Ruby on Rails, or juice Rails, which is a web development framework popular for the dynamic database backed web applications. Ruby has a reputation for having eloquent and rather strict syntax, but make it wonderful for data manipulation and processing. If you have ever wanted to create Application Programming Interfaces, or APIs.
